# Coding-With-Gen-AI
**My Initial prompt**
i want to create a simple gps where there's gonna be 9 houses shown by the yellow triangle, and each of the houses have their own number. there's also red rectangle which shown the enter and exit door. for example if i want to go from house number 1 to house number 2, you have to turn left when exit, then go straight then turn left and the enter door for house number 2 gonna be on your left. please make the coding using linked list.

Reflection
1. AI can create a really fast product with only simple prompt. I also learn that we need to always double check our product that generated by AI, because there is high chance that AI made some bugs and we need to make more prompt to fixed it.
2. You can create product faster and quicker. AI also offer problem solving if we're stuck.
3. AI can't code something that is too complex and if we didn't learn anything and only depend on AI, we won't be able to create something too complex.
4. In my opinion, know how to use and work with AI is Important but It is also important to improve your coding skill and not always depend on AI.
